# Dungeon Game
A procedurally generated roguelike dungeon crawler built in Python using Pygame.

This project is both a personal game development project, and a way to learn about the software engineering behind procedural generation, game architecture, and scalable design. 
Rather than relying on a game engine, the goal is to implement the core systems from the ground up in order to personally gain a deeper understanding of how games work internally.

## Current Progress
Currently, the game is in early development stages

### Implemented features:
- Basic Pygame loop
- Tilemap based room representation
- Configurable room generation modes
- Initial room generation templates
#### Current room types include:
- Start Room
- Corridor
- Placeholder implementations for:
  - Dungeon Rooms
  - Boss Rooms
  - Secret Rooms
  - Intersections
